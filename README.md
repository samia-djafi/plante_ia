# 🌿 PlanteIA

A research web platform that predicts the antibacterial activity of Algerian essential oils using AI — featuring MIC prediction, binary combination analysis, and genetic optimization.

> 3rd year AI Engineering project · UMMTO · 2025–2026

---

## Modules

- **MIC Prediction** — estimates the minimum inhibitory concentration from a GC-MS composition
- **Combination Analysis** — detects synergy / indifference / antagonism between two oils (FIC index)
- **Genetic Optimization** — finds the optimal Thyme / Rosemary / Mastic ratio over 50–100 generations
- **Library** — searchable database of 12 plants and 10 reference bacteria
---

> ⚠️ Demo mode — all predictions are simulated and must be validated experimentally.
